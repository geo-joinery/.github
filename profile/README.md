# geo-joinery

A "geo-joinery" is a GitHub repository that uses our workbench for JOIN operations on multiple Earth science datasets.
The geo-joinery organization builds the tools, you build a matchup dataset starting from [geo-joinery/start](https://github.com/geo-joinery/start).

But, we've only just started to get this project underway! Say "Hi!" in our [Discussions](https://github.com/orgs/geo-joinery/discussions) if you're interested in the concept!
